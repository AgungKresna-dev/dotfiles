# 🛠️ Dotfiles Managed by Chezmoi

Professional automated development environment configurations for Arch Linux and i3wm.

## 🚀 Quick Start

To deploy these dotfiles on a fresh machine, run the following one-liner:

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply USERNAME
```

## 📦 Stack
- **OS:** Arch Linux
- **WM:** i3wm
- **Manager:** [Chezmoi](https://www.chezmoi.io/)
